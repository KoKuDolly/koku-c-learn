## 数据类型
char          表示方法有：%c
int           表示方法有：%i %d %o %x %u 
                             %hd %ho %hx %hu 
                             %ld %lo %lx %lu
                             %lld %llo %llx %llu
float         表示方法有：%a %A %e %E %f %g %G %Le %Lf  科学计数法，double，long double，
pointer(指针) 表示方法有：%p
字符串        表示方法有：%s
%             表示方法有： %%
size_t %zd    这个作何解释
%n            这个作何解释
